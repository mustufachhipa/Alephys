
# Expense Tracker - Java Console App

## How to Compile and Run

1. Open Terminal / Command Prompt.
2. Navigate to the folder where files are extracted.
3. Compile the Java file:
   javac ExpenseTracker.java
4. Run the program:
   java ExpenseTracker

## Sample File Input Format (transactions.txt)
Each line should follow this format:
TYPE,YYYY-MM-DD,CATEGORY,AMOUNT

Example:
INCOME,2025-05-01,Salary,50000
EXPENSE,2025-05-03,Food,1200

Load this file using Option 3 in the app.
